# アバター通話（vtuber-call）

Webカメラの顔トラッキングでアバターを動かし、OBSの仮想カメラ経由で
**LINEビデオ通話のカメラ映像としてアバターを映す**ためのツール。

アバターは2方式に対応：

- **3Dモデル（VRM）**: VRoid Studio などで作った3Dキャラ
- **イラスト1枚（PNG/JPG）**: PNGtuber方式。好きな絵がそのまま動く

すべてブラウザ内（ローカル）で処理されるため、**通信課金なし・APIキーなし・回数制限なし**。

## 仕組み

```
Webカメラ
   │  映像
   ▼
MediaPipe Face Landmarker（顔468点+虹彩を検出）
   │  ランドマーク
   ▼
Kalidokit（顔の動き → アバターの骨・表情に変換）
   │  リグ情報
   ▼
three.js + three-vrm（VRMアバターを描画／胸から上のバストアップ構図）
   │  ブラウザ画面
   ▼
OBS ウィンドウキャプチャ → 仮想カメラ
   │  カメラ映像として出力
   ▼
PC版LINE（カメラに「OBS Virtual Camera」を選択）
```

## 反映される動き

- まばたき（左右別・ウィンク対応モデルなら片目も）
- 口パク（あ・い・う・え・お の母音形状）
- 首の傾き・向き（頭→首→背骨に自然に分配）
- 目線（黒目の動き）
- 呼吸モーション（待機中も生きてる感）
- 髪・服の揺れ（VRMのスプリングボーン物理）

手・全身のトラッキングは**意図的に省略**（バストアップ用途のため）。

## 使い方

### 1. 起動

`index.html` をブラウザ（Chrome推奨）でそのまま開く。

カメラ許可が出えへん場合はローカルサーバー経由で：

```powershell
cd vtuber-call
npx serve .
# → 表示されたURL（http://localhost:3000 など）を開く
```

### 2. アバターを読み込む

ファイルを画面に**ドラッグ&ドロップ**するだけ（または「ファイルを開く」）。
一度読み込んだアバターはブラウザに保存され、**次回起動時に自動復元**される。

#### 方式A: イラスト1枚（PNG/JPG）— PNGtuber方式

1. 画像をドロップすると設定画面が開く
2. **使う範囲をドラッグで囲む**（複数ポーズが並んだシート画像なら正面の絵だけ囲む。1枚絵ならそのまま「全体を使う」）
3. **左目 → 右目 → 口** の位置を順にクリックして「完了」
4. 背景は自動で透過される（うまくいかへんときはパネルの「背景を自動で透過する」をOFF）

動き：まばたき（絵の毛色・肌色に合わせたまぶたを自動描画）、口パク、首の傾き・向き、呼吸の揺れ。
ホイールで拡大縮小、ドラッグで位置調整。「範囲・目・口を再設定」ボタンでいつでもやり直せる。

#### 方式B: 3Dモデル（VRM）

- **とりあえず試す**: 「3Dサンプル」ボタン
- 自作は [VRoid Studio](https://vroid.com/studio)（無料）で作成し「VRMエクスポート」したファイルをドロップ
- **胸から上しか映らへんので、顔と髪と上半身の服だけ作り込めばOK**

### 3. 画面の調整

- マウスドラッグ／ホイールで構図を微調整（バストアップに自動フレーミング済み）
- 背景色はパネルから変更（グリーンバックにすればOBSでクロマキー合成も可能）
- **Hキー**でパネルとカメラ映像を非表示（OBSキャプチャ用のクリーン画面）

### 4. OBS設定

1. [OBS Studio](https://obsproject.com/ja)（無料）をインストール
2. ソース追加 →「ウィンドウキャプチャ」→ このブラウザのウィンドウを選択
3. 必要なら右クリック→「変換」→「画面に合わせる」でサイズ調整
4. 「仮想カメラ開始」ボタンを押す

### 5. LINE設定

1. **PC版LINE**（Windows版）を起動
2. 設定 → 通話 → カメラ →「**OBS Virtual Camera**」を選択
3. 友達とビデオ通話開始 → 相手にはアバターが映る

## スマホ1台で完結する方法（PC・OBS不要）

スマホはカメラの差し替え（仮想カメラ）ができへんので、代わりに**LINEの画面シェア機能**を使う。

1. このページを**インターネット上に公開**しておく（GitHub Pages等。後述）
2. スマホのLINEで友達と**ビデオ通話を開始**し、自分の**カメラをOFF**にする
3. 通話メニューから「**画面シェア**」を開始する
4. ホーム画面に戻り、ブラウザ（Chrome / Safari）でこのページのURLを開く
5. カメラを許可 →「ネコアバター」をタップ → 顔に合わせてアバターが動く
6. 相手には画面シェア越しに動くアバターが見える。**声は普段どおりLINEのマイクで届く**

ポイント：

- LINE側のカメラをOFFにしておくこと（ブラウザとカメラの取り合いになるため）
- 通知が画面シェアに映るので、おやすみモード推奨
- 付属のネコアバターはボタン1つで出る（切り抜き・目口の設定済み）
- 画面は自動で消灯せえへんようにしてある（Wake Lock対応ブラウザ）

## 内蔵ネコアバター

「ネコアバター」ボタンで、切り抜き済みの白ネコ（`cat_assets.js`、目・口の位置設定済み）が即座に使える。
元イラストの差し替えは `assets_work\` の手順（ポリゴン切り抜きスクリプト）を参照。

## URLパラメータ

| パラメータ | 例 | 効果 |
|---|---|---|
| `?ui=0` | `index.html?ui=0` | パネル非表示で起動（OBS用） |
| `?bg=00b140` | `index.html?bg=00b140` | 背景色指定 |
| `?vrm=URL` | `index.html?vrm=https://...` | VRMをURLから読み込み |

## トラブルシューティング

| 症状 | 対処 |
|---|---|
| カメラが起動しない | ブラウザのカメラ許可を確認。file://でダメなら `npx serve` で起動 |
| 首が左右逆に動く | パネルの「ミラー」チェックを切り替える |
| 首が上下逆に動く | パネルの「上下の動きを反転」をON |
| ブラウザを裏に回すとOBSの映像が止まる | ウィンドウの**最小化はNG**。他ウィンドウの裏に隠すのはOK。それでも止まるならChromeを `--disable-backgrounding-occluded-windows` フラグ付きで起動 |
| 口パクが反応しない | 顔が明るく正面に映るようにカメラ位置を調整 |
| 画像の背景透過で絵の一部まで消える | 「背景を自動で透過する」をOFFにして、透過済みPNGを用意してドロップする |
| まぶた・口の位置がズレてる | 「範囲・目・口を再設定」でクリックし直す |
| OBSにマウスカーソルが映る | ウィンドウキャプチャのプロパティで「カーソルをキャプチャ」をOFF |
